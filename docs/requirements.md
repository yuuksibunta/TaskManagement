# 要件定義書

## 基本情報

| 項目 | 内容 |
|------|------|
| アプリ名 | タスク管理アプリ（Trello風） |
| 利用者 | 自分1人 |
| 動作環境 | PC ブラウザ |
| 作成日 | 2026-04-21 |

---

## 機能一覧

### 1. カンバンボード

- ボードは1つ固定
- リストは以下の3列で固定
  - 「To Do」
  - 「進行中」
  - 「完了」
- カードをリスト間でドラッグ＆ドロップで移動できる

### 2. カード管理

- カードの作成・編集・削除ができる
- カードにタイトルとテキストメモを書ける

### 3. ユーザー認証

- メールアドレス＋パスワードでログイン・ログアウトができる
- 未ログイン時はボードを表示しない

### 4. データ保存

- カードの内容をサーバーに保存する
- どのPCからでもログインすれば同じデータを参照できる

---

## 非機能要件

| 項目 | 内容 |
|------|------|
| パフォーマンス | ページ読み込み 3秒以内 |
| 対応ブラウザ | Google Chrome のみ |
| セキュリティ | パスワードはハッシュ化して保存する |
| 可用性 | 学習用のため高可用性は不要 |
| バックアップ | 不要 |
| セッション | ブラウザを閉じたら自動ログアウト |
| データ上限 | 上限なし |

---

## スコープ外（今回は作らないもの）

- 複数ボード
- 画像・ファイル添付
- 期限・優先度設定
- 複数ユーザーでの共有
