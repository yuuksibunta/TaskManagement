# 機能要件

[← ドキュメント一覧に戻る](../README.md)

---

## 1. カンバンボード

- ボードは1つ固定
- リストは以下の3列で固定
  - 「To Do」
  - 「進行中」
  - 「完了」
- カードをリスト間でドラッグ＆ドロップで移動できる

## 2. カード管理

- カードの作成・編集・削除ができる
- カードにタイトルとテキストメモを書ける

## 3. ユーザー認証

- メールアドレス＋パスワードでログイン・ログアウトができる
- 未ログイン時はボードを表示しない

## 4. データ保存

- カードの内容をサーバーに保存する
- どのPCからでもログインすれば同じデータを参照できる
