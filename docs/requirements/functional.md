## 1. カンバンボード

- ボードは1つ固定
- リストは以下の3列で固定
  - 「To Do」
  - 「進行中」
  - 「完了」
- カードをリスト間・リスト内でドラッグ＆ドロップで移動できる
  - ドラッグ中はつかんでいるカードがカーソルに追従して表示される
- 各リストのカードを「優先度」または「期限」でソートできる
  - ソートボタンを再度押すと手動並び順に戻る

## 2. カード管理

- カードの作成・編集・削除ができる
- カードに以下の情報を設定できる
  - タイトル（必須）
  - メモ（任意）
  - 優先度：高 / 中 / 低（任意）
  - 期限（任意）
- 期限が過ぎたカードは赤バッジで強調表示する

## 3. データ保存

- カードの内容をサーバーに保存する
